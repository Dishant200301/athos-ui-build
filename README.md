# Athos

## Project info

**URL**: https://athoscollagen.com

## Getting Started

This project is built with Vite, React, TypeScript, shadcn-ui, and Tailwind CSS.

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:
   ```sh
   cd athos-ui-build
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```

## Building for Production

To create a production build with static prerendering:

```sh
npm run build
```

This will generate static HTML files in the `dist` folder.

## Technologies Used

- **Vite**: Build tool and dev server
- **React**: UI library
- **TypeScript**: Static typing
- **Tailwind CSS**: Utility-first CSS framework
- **shadcn-ui**: UI component collection
